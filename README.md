# 🌿 Rokacheat

**Rokacheat** is a Flutter-based mobile application developed as a mobile assignment project.  
The application demonstrates modern UI design, local database integration, mock data usage, and cloud image storage using Supabase.

---

## 📱 Overview

- Flutter mobile development
- Local data persistence using SQLite
- Cloud image storage integration
- Mock data implementation
- Navigation and authentication flow

The app supports offline data storage and retrieves images from Supabase Storage.

---

## 🚀 Features

- 🔐 Login & Register (Local)
- 🏠 Home Screen(Mock Data)
- 🔎 Search Screen
- 🛒 Cart Screen
- ⚙️ Settings Screen
- 📌 Bottom Navigation Bar
- 💾 Local Database using SQLite
- ☁️ Image Storage with Supabase

---
## 🗄️ Data Management

- Local Storage
    + Uses SQLite via sqflite
    + Stores user and app-related data locally
    + Supports offline access

- Mock Data
    + Static data used for UI development and testing
    + Helps simulate real backend data

- Supabase Storage
    + Images are uploaded and stored in Supabase
    + Image URLs are retrieved dynamically in the app
